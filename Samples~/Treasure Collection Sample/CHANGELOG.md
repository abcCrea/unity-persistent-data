# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-07-14

### Fixed
- Fixed the sample path so the Treasure Collection Sample can be imported correctly from the Unity Package Manager.

### Added
- Initial release.
- PersistentDataManager for primitive values using PlayerPrefs.
- JSON serialization for classes.
- Serializable dictionary support.
- Save version management.
- Save/load/delete events.
- Playable Treasure Collection Sample.
