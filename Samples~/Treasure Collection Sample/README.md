# Treasure Collection Sample

This sample demonstrates a basic gameplay persistence workflow using `abcCrea.PersistentData`.

## Controls

- **WASD**: Move
- **Arrow Keys**: Move

## Objective

Move through the map and collect rewards from active treasure chests.

Collected Sapphire, Ruby, and Emerald amounts are saved locally and loaded when the scene starts again.

## Demonstrated Features

- Saving and loading integers
- Trigger-based reward collection
- Weighted random rewards
- Timed treasure activation
- ScriptableObject configuration
- Runtime TextMeshPro UI updates

## Player Tag

The player GameObject uses Unity's built-in `Player` tag.

No custom tags need to be added to the project.
